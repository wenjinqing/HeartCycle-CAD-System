# HeartCycle 冠心病风险预测系统 - 项目总结

## 📋 项目概述

**项目名称**: HeartCycle 冠心病风险预测系统
**项目类型**: 毕业设计项目
**开发周期**: 2025.12 - 2026.02
**项目状态**: ✅ 已完成
**代码总量**: ~12,304行

## 🎯 项目目标

开发一个基于深度学习和机器学习的冠心病风险预测系统，为医疗机构提供智能化的疾病风险评估工具，包括患者管理、风险预测、报告生成、模型版本管理等完整功能。

## 🚀 部署方式

### Docker部署（推荐）
```bash
docker-compose up -d
```

### 手动部署
```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 前端
cd frontend
npm install
npm run build
```
