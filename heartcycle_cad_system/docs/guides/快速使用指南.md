# HeartCycle 系统快速使用指南

## 🚀 5分钟快速上手

### 第一步：启动系统

```bash
# 启动后端
cd heartcycle_cad_system
python scripts/start_backend.py

# 启动前端（新终端）
cd frontend
npm run dev
```

访问：http://localhost:5173

---

### 第二步：登录系统

**测试账号：**
- 用户名：`admin`
- 密码：`password123`

> 更多测试账号见 `测试数据/test_users.json`

---

### 第三步：体验核心功能

#### 功能1：风险监测（最简单）

1. 点击首页"立即开始分析"
2. 填写患者信息（年龄、性别、BMI等）
3. 点击"开始分析"
4. 查看风险评估结果

**进阶：** 上传心电数据
- 点击"上传HDF5文件"
- 选择 `测试数据/ecg_samples/sample_001.h5`
- 系统自动提取HRV特征

---

#### 功能2：H5快速训练（推荐）

**路径：** 首页 → H5快速训练

**步骤：**
1. 上传H5文件（`测试数据/h5_training_samples/` 目录下所有.h5文件）
2. 系统自动识别标签（从SubjectMetadata.csv）
3. 选择模型类型（推荐：随机森林）
4. 点击"开始训练"
5. 等待训练完成（约2-5分钟）

**重要：** 确保 `SubjectMetadata.csv` 与H5文件在同一目录！

---

#### 功能3：批量预测

**路径：** 导航栏 → 批量预测

**步骤：**
1. 上传 `测试数据/batch_predict_sample.csv`
2. 选择预测模型
3. 点击"开始预测"
4. 导出结果

---

#### 功能4：论文实验（完整流程）

**路径：** 导航栏 → 论文实验系统

**适合：** 研究人员进行完整实验

**步骤：**
1. **数据预处理：** 上传 `测试数据/experiment_dataset.csv`
2. **特征工程：** 自动特征选择
3. **模型训练：** 训练RF、XGBoost、LightGBM
4. **结果分析：** SHAP分析、性能对比

**时间：** 约10-15分钟

## ⚠️ 常见问题

### Q1: 上传H5文件后提示"找不到元数据"
**A:** 确保 `SubjectMetadata.csv` 与H5文件在同一目录

### Q2: 训练任务一直显示"准备中"
**A:** 检查后端服务是否正常运行，查看控制台日志

### Q3: 权限不足无法访问某些页面
**A:** 使用 `admin` 账号登录可访问所有功能

### Q4: CSV文件中文乱码
**A:** 确保CSV文件使用UTF-8编码保存
