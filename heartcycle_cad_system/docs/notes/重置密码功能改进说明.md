# 重置密码功能改进说明

## 改进内容

### 原功能
- 管理员点击"重置密码"按钮
- 密码自动重置为固定的 `password123`
- 无法自定义密码

### 新功能
- 管理员点击"重置密码"按钮
- 弹出输入框，管理员可以自定义新密码
- 密码长度至少6位
- 支持密码强度验证

## 修改文件

### 1. 前端 - `frontend/src/views/admin/Users.vue`

**改进：** 使用 `ElMessageBox.prompt` 替代 `ElMessageBox.confirm`

```javascript
// 修改前：固定密码
await ElMessageBox.confirm(
  `确定要重置用户密码吗？新密码将重置为 "password123"`,
  '确认重置密码'
)

// 修改后：自定义密码
const { value: newPassword } = await ElMessageBox.prompt(
  `请输入用户 "${user.username}" 的新密码`,
  '重置密码',
  {
    inputPattern: /^.{6,}$/,
    inputErrorMessage: '密码长度至少6位',
    inputPlaceholder: '请输入新密码（至少6位）',
    inputType: 'password'
  }
)
```

### 2. 前端 - `frontend/src/services/api.js`

**改进：** 传递新密码参数

```javascript
// 修改前
resetUserPassword(userId) {
  return api.post(`/auth/users/${userId}/reset-password`)
}

// 修改后
resetUserPassword(userId, newPassword) {
  return api.post(`/auth/users/${userId}/reset-password`, {
    new_password: newPassword
  })
}
```

### 3. 后端 - `backend/app/api/v1/auth.py`

**改进：** 接收并验证新密码

```python
async def reset_user_password(
    user_id: int,
    request: Request,
    password_data: dict,  # 新增：接收请求体
    current_user: User = Depends(require_admin),
    db: Session = Depends(get_db)
):
    # 获取新密码
    new_password = password_data.get('new_password')

    # 验证密码
    if not new_password:
        raise HTTPException(status_code=400, detail="新密码不能为空")

    if len(new_password) < 6:
        raise HTTPException(status_code=400, detail="密码长度至少6位")

    # 重置密码
    auth_service.reset_user_password(user_id, new_password)
```

## 密码验证规则

### 前端验证
- ✅ 密码长度至少6位
- ✅ 不能为空
- ✅ 实时验证（输入时）

### 后端验证
- ✅ 密码长度至少6位
- ✅ 不能为空
- ✅ 使用pbkdf2_sha256加密存储

## 安全改进

### 优点
1. **灵活性** - 管理员可以根据安全策略设置不同强度的密码
2. **安全性** - 避免所有重置密码都使用相同的默认密码
3. **可追溯** - 审计日志记录重置操作（不记录密码明文）
4. **用户体验** - 管理员可以设置易记的临时密码

### 建议
1. **通知用户** - 重置密码后，建议通过邮件或其他方式通知用户
2. **强制修改** - 建议用户首次登录后立即修改密码
3. **密码策略** - 可以进一步增强密码强度要求（大小写、数字、特殊字符）

## 可选增强功能

如果需要更强的密码策略，可以修改验证规则：

### 前端增强
```javascript
inputPattern: /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}$/,
inputErrorMessage: '密码至少8位，包含大小写字母和数字'
```

### 后端增强
```python
import re

def validate_password_strength(password: str) -> bool:
    """验证密码强度"""
    if len(password) < 8:
        return False
    if not re.search(r'[a-z]', password):
        return False
    if not re.search(r'[A-Z]', password):
        return False
    if not re.search(r'\d', password):
        return False
    return True
```
